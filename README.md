**Your Health. Their Life.**

> "Symbi connects your physical health to a virtual life. This app gamifies calorie tracking by linking your nutrition to a digital pet: when you eat well, they thrive. Designed to make portion control engaging, Symbi turns self-care into a shared responsibility."

## Tech Stack

- **Frontend:** React Native (Expo) + TypeScript
- **Backend:** Node.js + Express
- **Database:** Supabase (PostgreSQL)
- **API:** OpenFoodFacts

## Structure

- `/client` - The Mobile Application
- `/server` - The Backend API
